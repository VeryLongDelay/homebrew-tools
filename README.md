# VeryLongDelay's Homebrew Tools

This repository contains the Homebrew tools for VeryLongDelay.

## Installation

Install homebrew if you haven't already:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Add the tap:

```bash
brew tap VeryLongDelay/tools
```
